# Part 1: Functional Programming - Remove Duplicates

## Approach
Uses Python's `functools.reduce` to build a new list of unique elements while preserving the original order of first occurrences. The function is pure — it has no side effects and never modifies the input list.

## How to Run
```bash
cd part1-functional
python remove_duplicates.py
```

Expected output: `All tests passed!`

## Requirements
- Python 3.7+
- No external dependencies
